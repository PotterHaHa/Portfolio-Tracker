API 資料來源說明

TWSE 上市 台灣證券交易所 OpenAPI
收盤價：https://openapi.twse.com.tw/v1/exchangeReport/STOCK_DAY_AVG_ALL
殖利率：https://openapi.twse.com.tw/v1/exchangeReport/BWIBBU_d
適用 4~5 碼上市代號（0050、00878、00713 等）。
更新時間：前一個交易日的盤後資料，通常隔天凌晨更新完成。免費，無需 API Key。
TPEx 上櫃 證券櫃檯買賣中心 OpenAPI
收盤價：https://www.tpex.org.tw/openapi/v1/tpex_mainboard_daily_close_quotes
適用帶英文字母的 6 碼代號（00772B、00400A 等上櫃 ETF）。
更新時間：當日盤後，帶民國日期參數（自動帶入）。免費，無需 API Key。
殖利率說明
官方殖利率（TWSE BWIBBU_d）= 近一年股利 ÷ 收盤價，僅適用上市股。月配型 ETF 若配息不足 12 次則偏低。

自填殖利率建議做法：去各 ETF 官網查最近 12 個月配息加總，手動除以現價計算，最準確。

CORS 說明：這個工具以本機 HTML 或 GitHub Pages 開啟時可正常呼叫 API。若嵌在其他網頁（如 claude.ai）中則因瀏覽器跨域限制可能無法抓取。
