使用說明

更新股價手動輸入收盤價
在「持股總覽」的現價欄位直接輸入收盤價，市值、損益、殖利率會即時重新計算。

去哪裡查股價：
・上市股（0050、00878、00400A、00713）→ 證交所
・上櫃股（00772B）→ 櫃買中心
・或用 玩股網、Yahoo 股市等查當日收盤價。

提醒：輸入價格後若要跨裝置保存，記得按「儲存書籤」更新你的書籤 URL。
加密方式AES-GCM 256-bit（瀏覽器原生）
原理：你的持股資料在按下「儲存書籤」時，以你輸入的密碼透過 PBKDF2 衍生金鑰，用 AES-GCM 256-bit 加密，編碼為 Base64 後附加在 URL 的 # 後面。

URL Hash 的特性：# 後面的內容瀏覽器不會傳給伺服器，因此 GitHub 完全看不到你的任何資料。

安全說明：密碼只存在你的記憶中，加密在本地完成，沒有任何資料上傳。就算有人拿到你的 URL，沒有密碼也無法解密。
跨裝置使用Chrome 書籤同步
用 Chrome 登入 Google 帳號並開啟書籤同步後，「儲存書籤」存下的 URL 會自動同步到你所有裝置。換電腦時開同一個書籤、輸入密碼即可還原。

習慣養成：每次改持股或更新股價後，按一次「儲存書籤」並覆蓋更新舊書籤，確保資料是最新的。
